# Netflix-Data-Cleaning-Internship
Task 1 Summary: Netflix Data Cleaning
Objective: Cleaned the raw Netflix dataset to prepare it for analysis.

Changes Made:

Handled Missing Values: Filled missing directors and cast with "Unknown" and filled missing countries with the most frequent value.

Removed Duplicates: Scanned the dataset and removed any duplicate rows to ensure data quality.

Standardized Formats: Converted the date_added column from text to a standard Date format.

Renamed Columns: Converted all column headers to lowercase for uniform coding.

Data Validation: Removed rows with null values in critical columns like rating and duration.
